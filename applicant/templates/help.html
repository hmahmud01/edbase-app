Guidline to upload A file

  • First Create A Teacher from Teachers Link
  • First Create A Subject from Subjects Link
  • Assign that teacher to that subject
  • Now redirect to Upload Content Page Link

Guidline to Session and Batch Feather

  • Create a Session
  • Create a Batch
  • Assign this session and batch to the student as he is expecting (This feature coming soon)
  • Follow these steps in any order but to assign a session and batch please create them first
  • N.B. > Do not delete the first session and batch which is already created with none tag
